Why use semantic elements?

Top Five Reasons

click to view

Meaningful

html should provide meaning when looked at

Div and Span

div and span can be use sparingly but can be overused very quickly, providing no meaning to large documents of code

Screen Reader Friendly

screen readers rely on semantically-rich HTML to process web pages and help users with low vision navigate them. Semantic tags tell screen reader users where they are and how they can interact with the current page

Search Engines

search engines can identify what the website is about eaiser

makes crawl bots happy, because non semantic webpages is usually ignored by search engines

Developer Sanity

easier to work on when tags give more meaning to user trying to understand how pieces are interacting
